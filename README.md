# Quora Clone Project

<br>
It is a clone of Quora WebApp.
<br>
# Pre-requisites
<br>
Install Node.js.
<br>
# Getting Started
<br>
- Clone the repository
```
    git clone "https://github.com/Amaan-12-git/Quora-CLone.git"
```
- Intsall Dependencies
```
    npm install
```
- Build and run the project
```
npm start
```
